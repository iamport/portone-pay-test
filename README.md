# PortOne SDK 놀이터

주소: <https://sdk-playground.portone.io/>

PortOne의 JS SDK를 시험해볼 수 있는 놀이터입니다.

결제, 본인인증 등을 실행할 수 있습니다.

## 개발하기

### 준비물

- Node.js: <https://nodejs.org/en/>
- pnpm: <https://pnpm.io/>

## 개발 서버 실행하기

```sh
pnpm install
pnpm dev
```

## 배포하는 법

수정사항을 이 저장소의 `main` 브랜치에 병합하면 Vercel을 통해서 자동으로
프로덕션에 배포됩니다.
